# PolyMath-3D
PolyMath-3D - это приложения для работы с трехмерной математикой и матричными преобразованиями, построенная с использованием архитектурных принципов и типов данных движка Bevy.

Проект создан для того, чтобы расширить стандартные возможности работы с трансформациями в Bevy или предоставить удобную обертку для сложных геометрических вычислений, которые часто требуются при разработке систем на базе ECS (Entity Component System).

🛠 Особенности проекта
Bevy Integration: Полная совместимость с типами Transform, GlobalTransform и внутренними структурами данных Bevy.

Матричные операции: Углубленная реализация манипуляций с матрицами 4×4 для специфических задач (например, кастомные проекции или сложные иерархии объектов).

Векторная оптимизация: Использование мощностей glam (стандарт Bevy для математики) с добавлением высокоуровневых абстракций.

Системный подход: Код написан с учетом производительности и удобной интеграции в App через плагины.

🚀 Почему Bevy?
Bevy предоставляет отличный фундамент, но PolyMath-3D идет дальше, позволяя:

Визуализировать математические процессы "под капотом" движка.

Создавать кастомные системы для процедурной генерации мешей (например, с использованием Perlin Noise или алгоритмов триангуляции).

Работать с низкоуровневыми шейдерными данными напрямую.
